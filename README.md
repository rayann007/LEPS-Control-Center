# 🖥 LEPS Control Center

> PowerShell WPF administration tool — Leoni Wiring Systems IT Dept.

## Description
Interface graphique centralisée pour la gestion des serveurs d'impression
LEPS sur 3 sites industriels (LTN1, LTN2, LTN4).

## Fonctionnalités
- 🌐 Gestion multi-site : LTN1 / LTN2 / LTN4
- 🖨 Centralisation de 8 serveurs d'impression (200+ imprimantes)
- ⚙️ Installation, configuration et démarrage LEPS
- 📋 Supervision des files d'impression (In Process / Idle)
- 📷 Générateur de QR Code intégré
- 🔒 Accès direct Cisco ISE NAC
- 🚀 Réduction du temps de diagnostic de 40%

## Lancement
```powershell
powershell -ExecutionPolicy Bypass -File "LEPS_Control_Center.ps1"
```

## Stack
- PowerShell 5.1+
- WPF (Windows Presentation Foundation)
- XAML UI

## Contexte
Développé en environnement industriel chez **Leoni Wiring Systems Tunisia**
dans le cadre de l'optimisation des opérations IT sur 3 sites.
